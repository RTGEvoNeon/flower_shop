---
description: Создать новую модель со всеми необходимыми файлами
---

Создай полноценную модель со следующим:

1. **Миграция**
   - Все необходимые поля
   - Индексы для часто используемых полей
   - Foreign keys с правильными constraints
   - Метод down() для отката

2. **Модель**
   - $fillable массив
   - $casts для типизации
   - Relationships (если есть)
   - Scopes для часто используемых запросов
   - Accessors/Mutators (если нужны)
   - PHPDoc комментарии для IDE

3. **Form Request**
   - StoreRequest для создания
   - UpdateRequest для обновления
   - Правила валидации
   - Сообщения на русском

4. **Resource Controller**
   - Все CRUD методы
   - Использование Form Requests
   - Редиректы с сообщениями
   - Try-catch для обработки ошибок

5. **Роуты**
   - Route::resource() с именем
   - Middleware если нужен

Следуй PSR-12 и используй строгую типизацию.
