description: Проверить SEO оптимизацию страницы
---

Проверь SEO оптимизацию текущей страницы:

1. **Meta теги**
   - [ ] Title (50-60 символов, уникальный, с ключевыми словами)
   - [ ] Description (150-160 символов, призыв к действию)
   - [ ] Keywords (5-10 релевантных слов)

2. **Open Graph**
   - [ ] og:title
   - [ ] og:description
   - [ ] og:image (минимум 1200x630px)
   - [ ] og:url
   - [ ] og:type

3. **Schema.org разметка**
   - [ ] JSON-LD присутствует
   - [ ] Корректный тип (LocalBusiness, Product, Article)
   - [ ] Все обязательные поля заполнены

4. **Контент**
   - [ ] Один H1 с ключевым словом
   - [ ] Иерархия заголовков (H1→H2→H3)
   - [ ] Alt теги у всех изображений
   - [ ] Ключевые слова в первом абзаце
   - [ ] Внутренние ссылки

5. **Технические аспекты**
   - [ ] Canonical URL указан
   - [ ] URL читаемый (ЧПУ)
   - [ ] Изображения оптимизированы
   - [ ] Loading="lazy" для изображений вне видимости
   - [ ] Mobile-friendly

6. **Локальное SEO (Брянск)**
   - [ ] Упоминание города в title
   - [ ] Упоминание города в description
   - [ ] Упоминание города в H1 или контенте

Проанализируй и предложи улучшения.
