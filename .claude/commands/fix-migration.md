---
description: Проверить и исправить миграцию
---

Проверь миграцию на соответствие стандартам:

1. **Структура**
   - [ ] Метод up() присутствует
   - [ ] Метод down() присутствует и корректен
   - [ ] Используется правильный тип таблицы (create/table)

2. **Поля**
   - [ ] ID поле: bigIncrements('id') или id()
   - [ ] Timestamps: timestamps() или created_at/updated_at
   - [ ] Nullable только где действительно нужно
   - [ ] Дефолтные значения указаны где возможно
   - [ ] after() используется для правильной позиции

3. **Foreign Keys**
   - [ ] Правильное именование: {table}_id
   - [ ] Constraint указан
   - [ ] onDelete() и onUpdate() настроены
   - [ ] Индекс автоматически создается

4. **Индексы**
   - [ ] Индексы на foreign keys
   - [ ] Индексы на поля для поиска (slug, email)
   - [ ] Индексы на поля для сортировки
   - [ ] Уникальные индексы где нужно

5. **Комментарии**
   - [ ] Сложные поля имеют комментарии на русском
   - [ ] Назначение таблицы понятно

6. **Безопасность**
   - [ ] Нет жестко заданных данных
   - [ ] Sensitive данные не в миграции

Проверь и предложи исправления.
