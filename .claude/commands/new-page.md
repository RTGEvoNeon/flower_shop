---
description: Создать новую SEO-оптимизированную страницу
---

Создай новую страницу со следующими шагами:

1. Создай контроллер (или добавь метод в существующий)
2. Создай Blade шаблон со всеми SEO элементами:
   - Meta title, description, keywords
   - Open Graph теги
   - Schema.org разметку (JSON-LD)
   - Семантическую HTML разметку
   - Alt теги для изображений
3. Добавь роут в web.php
4. Добавь ссылку в навигацию (если нужно)
5. Проверь мобильную версию (responsive)

Не забудь:
- Использовать ключевые слова "доставка цветов Брянск" в мета-тегах
- Добавить canonical URL
- Использовать семантические теги (article, section, nav)
- Добавить страницу в sitemap позже
