---
description: Оптимизировать изображения на странице
---

Проанализируй использование изображений на странице:

1. **Проверь атрибуты**
   - [ ] Alt текст присутствует и описательный
   - [ ] Width и height указаны (предотвращает layout shift)
   - [ ] Loading="lazy" для изображений вне первого экрана
   - [ ] Title атрибут (опционально)

2. **Формат и размер**
   - Предложи использовать WebP формат
   - Проверь, нет ли слишком больших изображений
   - Рекомендуемые размеры:
     - Превью товаров: 400x300px
     - Детальное фото: 1200x900px
     - OG изображение: 1200x630px
     - Иконки: SVG или оптимизированный PNG

3. **SEO оптимизация**
   - Alt текст содержит ключевые слова
   - Имя файла описательное (не IMG_1234.jpg)
   - Использовать схему: тип-назначение-номер.webp
     Пример: buket-roz-krasnye-001.webp

4. **Техническое**
   - Используется srcset для разных экранов (если нужно)
   - Изображения загружаются из правильной директории
   - CDN используется (если настроен)
