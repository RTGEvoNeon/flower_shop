description: Провести code review по стандартам проекта
---

Проведи code review текущих изменений:

1. **Стиль кода**
   - [ ] Соответствует PSR-12
   - [ ] Строгая типизация используется
   - [ ] Типы возвращаемых значений указаны
   - [ ] Комментарии на русском языке
   - [ ] Названия на английском

2. **Laravel best practices**
   - [ ] Использован правильный паттерн (Controller/Service/Model)
   - [ ] Нет N+1 проблем (используется eager loading)
   - [ ] Валидация через Form Requests
   - [ ] Используются именованные роуты
   - [ ] Route Model Binding где возможно

3. **Безопасность**
   - [ ] Нет SQL инъекций
   - [ ] Нет XSS уязвимостей
   - [ ] Входные данные валидируются
   - [ ] CSRF защита для форм
   - [ ] Конфиденциальные данные не в коде

4. **Производительность**
   - [ ] Запросы оптимизированы
   - [ ] Нет дублирования запросов
   - [ ] Используется кеширование где нужно
   - [ ] Изображения оптимизированы

5. **SEO (если это страница)**
   - [ ] Все SEO теги присутствуют
   - [ ] Schema.org разметка корректна
   - [ ] Семантическая разметка
   - [ ] Alt теги для изображений

6. **Тестируемость**
   - [ ] Код легко тестировать
   - [ ] Нет жесткой связанности
   - [ ] Используется DI

Предоставь feedback и предложения по улучшению.
