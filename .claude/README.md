# Claude Code Configuration

Эта директория содержит конфигурацию и правила для работы с Claude Code в VS Code.

## Файлы

### `instructions`
Основной файл с подробными правилами разработки проекта Mindale.
Включает:
- Стандарты кода
- Laravel best practices
- SEO требования
- Безопасность
- Git workflow
- Специфика проекта

### `commands/`
Директория с кастомными командами для быстрого выполнения типовых задач.

#### Доступные команды:

- `/new-page` - Создать новую SEO-оптимизированную страницу
- `/seo-check` - Проверить SEO оптимизацию текущей страницы
- `/create-model` - Создать полноценную модель со всеми файлами
- `/review-code` - Провести code review по стандартам проекта
- `/optimize-images` - Оптимизировать изображения на странице
- `/fix-migration` - Проверить и исправить миграцию

## Как использовать

1. **Правила проекта** автоматически применяются при работе с Claude Code
2. **Кастомные команды** вызываются через `/` в чате Claude Code
   Например: `/new-page` запустит процесс создания новой страницы

## Добавление своих команд

Создайте файл `.claude/commands/your-command.md`:

```markdown
---
description: Краткое описание команды
---

Детальная инструкция что должен сделать Claude.
Можно использовать чеклисты, примеры кода, и т.д.
```

После создания команда будет доступна как `/your-command`

## Обновление правил

Просто отредактируйте файл `instructions` - изменения применятся автоматически.

## Игнорирование правил

Если для конкретной задачи нужно отклониться от правил, явно укажите это в запросе:
> "Игнорируя стандартные правила, сделай..."

Но делайте это только когда действительно необходимо!
